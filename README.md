Multilingual category generator for [Hexo](http://hexo.io/).

## Installation

``` bash
$ npm install hexo-generator-multilingual-category --save
```

## Options

``` yaml
category_generator:
  per_page: 10
```

- **per_page**: Posts displayed per page. (0 = disable pagination).
